What is the ID system?

The ID system is a method of indexing the stations in our database. Each station is assigned a unique ID that consists of a combination of letters and numbers, we use for exemple S for Slavic if its Russian or Slavic language and we add the frequency the station has been discovered/heard for the first time. The ID is used to identify the station and to link it to its corresponding page in the wiki.

ID prefix Description
S Slavic station, usually Russian but not always.
P Pirate station.
E English station.
F French station.
D Dutch station.
CH Chinese station.
UNID / U Unidentified station. (stations with unknown or unclear identification)

Other ID systems

In addition to the Signabroam ID system, there are other ID systems used for Digital stations, for exemple Morse stations, and other Modem stations.
The difference is the numbers is not a frequency but a custom ID given.

ID prefix Description
MO Station using Morse code.
MD Station using a specific modem for digital communication.
TTY Station using TTY (Teletypewriter) communication. (for exemple RTTY, NAVTEX etc...)
NS NS - Numbers Stations, have multiples format, it use NS + the country code + the Discovery number + the format, for exemple for the FL numbers station, it was the first discovered and was using morse code the format is NS0001-MO etc...
TS TS - Time Stations, the format is pretty simple, it use TS + Station Name.
US-MIL US-MIL - United States Military stations, the format is US-MIL + Station Name.

About Patron79

Name: Patron79
Callsigns: Patron79, Bars16
Signabroam ID: S4790
Frequencies 4790 kHz
Status Inactive
Voice Male, Female (Two way monolith communication)
Mode USB
Location Russia
Known counterpart stations S28, S30, S32
Description Another russian military network, which transmits Monolyth type messages (signals). However, this network requires receivers confirmation.
Page of voice messages archives at : https://priyom.org/military-stations/russia/s4790/.

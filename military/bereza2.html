About Bereza2

Name: Bereza2
Callsigns: Bereza2
Frequencies 5182 kHz
Status Inactive
Voice Male, Female
Mode USB
Location 336th Russian Radio technical regiment, Orel, Russia
Known counterpart stations Zaliv52
Description Bereza2 is a station from the southern part of Russia. The station was running a channel marker and monolit type messages. The use is uncertain, not much information known about it.

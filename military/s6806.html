About The T Marker N°2

Name: The T Marker N°2
Callsigns: RHPW, Proz0 (Not sure)
Signabroam ID: S6806
Frequencies 6806 kHz
Status Active
Voice Male, Female
Mode USB
Location Somewhere in Leningrad, Russia
Known counterpart stations The T Marker
Description S6806 is a new Russian T Marker station appeared the 29.06.2026, the purpose is still currently unknown, the station transmit bunch of activity such as Test Counts, Net check (without callsign), and some other unknown activity...
The station use the same Marker as 4325 kHz and might be linked to the T Marker station, but this is not confirmed yet. As we know the markers are synchronised...

About CIS-12

Name: CIS-12 (AT-3004D / AT-3104D)
Frequencies Multiple
Status Active
Voice Male, Female
Mode USB/LSB/DSB/AM, QPSK12
Location Russia
Description AT-3004D and AT-3104 are nearly always seen transmitting using the 12-carrier PSK waveform. The supported data rates are 1200 bit/s (using BPSK) and 2400 bit/s (using QPSK).
More details at : https://www.sigidwiki.com/wiki/CIS-12.

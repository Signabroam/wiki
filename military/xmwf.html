About The Russian XM Whales Feedback

Name: The Russian XM Whales Feedback
Callsigns: N/A
Signabroam ID: U3623
Frequencies 3623, 3799, 5173, 5189 kHz
Status Inactive
Voice N/A
Mode USB
Location Russia
Known counterpart stations Northern Russian Network Southern Russian Network Kerro Pip
Description The Russian XM Whales Feedback is a mysterious shortwave station that has been observed on 3623, 5173, and 5189 kHz. It is believed to be associated with Russian military communications, but its exact purpose remains unknown. The station has been inactive for some time, and there is limited information available about its operations.
This stations was transmitting music-loop like the other stations sited in the Counterpart Stations.

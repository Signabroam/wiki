About Unit 32982

Name: Unit 32982
Callsigns: КДП, ЗСКП, РСБН, ДПРМ, БПРМ РСП
Signabroam ID: U32982
Frequencies Unknown
Status Unknown
Voice Male, Female
Mode USB
Location Unit 32982 "BSiRTO" Communication Hub ("Central AirField Radio Technical Batalion"), Rostov-on-Don, Russia
Known counterpart stations Unknown
Description Unit 32982 is a unit of the Russian Armed Forces responsible for the operation of the ZSKP (ЗСКП) station, which is a station used for communication with airplanes. The unit is also responsible for the operation on Russian Air Forces, which is used for communication with planes. The unit is based in Rostov-on-Don, Russia.

About S4467

Name: Slavic Call: S4467
Callsigns: VOLNUShKA-7, BUShLAT-11
Frequencies 4467, 4639
Status Inactive
Voice Male, Female
Mode USB
Location Sevastopol, Russia
Known counterpart stations S32, S30
Description A control network believed to be for 31st Air defence division from Sevastopol. Working under S32 (4th Air Defence army) and S30 (South District) networks.
Sending both network tests and control and command signals (monolith and usor).
Last heard in end 2017.
Archive page of voice messages at : http://priyom.org/military-stations/russia/s4467.

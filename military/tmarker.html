About The T Marker

Name: The T Marker
Callsigns: Osnova48, Ryabina68, Beryova20, K66Z, RHPW
Signabroam ID: S4524
Frequencies 4524, 4182, 4325
Status Active
Voice Male, Female
Mode USB
Location Near Kirov, Russia
Known counterpart stations N/A
Description "T-Marker" is a Russian military commandment network serving the Western Military District. It broadcasts on 4325 kHz round the clock.

While no other traffic is being sent, the station emits its signature channel marker, which consists of intermittent tone dashes akin to the Morse code letter "T". The traffic consists mainly of "Monolith" format messages addressed to РХПВ (RHPW).
Page of voice messages at : https://priyom.org/military-stations/russia/t-marker.

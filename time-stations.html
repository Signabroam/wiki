Welcome to the World wide Time Stations Wiki! This wiki is dedicated to providing information and resources about world wide time stations. Here you can find information about various time stations around the globe.


World wide Time Stations

Note about the modes:
AM, DSB = Audio + Ticks - Amplitude Modulation, Double Sideband
AMN = Narrow standard marker - Amplitude Modulation, Narrowband
FSK = Audio Shift - Frequency Shift Keying
ASK = Carrier reduced - Amplitude Shift Keying
PWM = Pulse Width Modulation/Encoding
OOK = On-Off Keying
PSK = Phase Modulation - Phase Shift Keying
CW Ticks = Pulse type morse code - Continuous Wave Ticks

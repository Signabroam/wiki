About USAF Global High Frequency System (GHFS)

Name: USAF Global High Frequency System (GHFS)
Signabroam ID: US-MIL-USAF-GHFS
Frequencies Same bands as HFGCS
Status Active
Mode USB, ALE
Location Worldwide
Description GHFS is often described as the modern successor to HFGCS, using the same high-frequency infrastructure for USAF strategic and command communications.

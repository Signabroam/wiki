About the HF Global Communication System (HFGCS)

Name: HF Global Communication System (HFGCS)
Callsigns: Mainsail, Andersen, Andrews, Ascension, Croughton, Diego Garcia, Elmendorf, Hickam, Lajes, Offutt, Puerto Rico, Sigonella, West Coast, Yokota
Signabroam ID: US-MIL-HFGCS
Frequencies 4724, 8992, 11175, 15016
Status Active
Voice Male, Female
Mode USB
Location Worldwide
Description The High Frequency Global Communications System (HFGCS) is the HF communication system of the United States Air Force.

The HFGCS has 13 ground stations near U.S. air bases around the world, all of which are remotely controlled from the Network Control Station at the Andrews Air Force Base.
See priyom.org HFGCS map at https://priyom.org/media/221924/hfgcs-map.html for more information about the locations of the stations.

For more global inforations visit the Priyom.org HFGCS page at : https://priyom.org/military-stations/united-states/hfgcs.

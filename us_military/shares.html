About SHARES

Name: Shared Resources HF Radio Program (SHARES)
Signabroam ID: US-MIL-SHARES
Frequencies Multiple HF frequencies
Status Active
Mode USB
Use Federal emergency communications, FEMA, DHS, government agencies
Description SHARES is an interagency HF communications program that allows federal agencies to coordinate during emergencies over shared HF resources.

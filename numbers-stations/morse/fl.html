About the FL Numbers Station

Name: FL Numbers Station
Call: FL - Tour Eiffel
ENIGMA ID: FL
Signabroam ID: NS3301-MO
Frequencies Unknown
Status Inactive
Type Morse
Mode CW
Location Eiffel Tower, Paris, France
Description The FL Numbers Station, also known as the "Tour Eiffel" station, was a mysterious shortwave radio station that transmitted Morse code messages. It was the first ever numbers station identified.
More informations can be found on numbers-stations.com.

About the English Man (E06)

Name: English Man
ENIGMA ID: E06
Signabroam ID: NS0706-E
Frequencies See Priyom.org for the schedule of E06.
Status Active
Voice Male
Mode USB (with Carrier)
Location Moscow, Russia
Orenburg, Russia
Smolensk, Russia
Chita, Russia
Description E06 is a Numbers Station operated by the operator Russian 6.
E06 format is a voice reading of numbers in English with 3 different format:
- Null: 3 Digit ID spelled thrice (during 4 minutes), and then 00000 to end the transmission.
- Traffic, single message: 3 Digit ID (during 4 minutes), then 3 digit group spelled twice, one Group count also spelled twice, 5 digit paired groups, 3 digit group spelled twice, a group count spelled twice, and then 00000 to end the transmission.
- Traffic, two message: 3 Digit ID (during 4 minutes), then 3 digit group spelled twice, one Group count also spelled twice, 5 digit paired groups, 3 digit group spelled twice, 3 Digit ID (during 1 minute), 3 digit group spelled twice, one group count spelled twice, 5 digit paired groups, 3 digit group spelled twice, group count spelled twice, a group count spelled twice, and then 00000 to end the transmission.

Informations from ENIGMA: 
E06 OM AM (English Man) Usually H+0, H+30 M14 / M24
Stays on UTC
Times/freqs vary seasonally, up to date charts given in Newsletters
High pitched start tone.
Null message format:-
“ nnn x3 00000 ” for 4 mins.
Message format:-
3f call for 4mins
preamble – 3f DK x2, 2/3f GC x 2.
Msg 5F grps,
Ends – DK x 2, GC x 2, 00000 (fast/slow zeros)
Note that the first 8 figs of TX has NO repeat figs.
(returned to the original voice early 2001, with distinctive pronunciation of 3 & 8)
(additional voice from Sept 03, with no perceived distinctive pronunciations )
E06a OM AM, 2 group commencing with 11111 stutter group
E06b OM AM, 3F ID with 5F group in call, 2 messages

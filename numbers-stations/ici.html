About the ICI Numbers Station

Name: ICI Numbers Station
Call: ICI
ENIGMA ID: ICI
Signabroam ID: NS0002-MO
Frequencies Unknown
Status Inactive
Type Morse
Mode CW
Location Cotona, Italy
Description The ICI Numbers Station, also known as the "Cotona" station, was a mysterious shortwave radio station that transmitted Morse code messages. It was one of the early numbers stations identified.
More informations can be found on numbers-stations.com.
